# Mercadito-Teteu-Sasa
Este sistema é uma aplicação para gerenciamento de cadastro de clientes, produtos, compras e vendas, voltada para mercados ou pequenas lojas. O objetivo é facilitar o controle e o registro das operações comerciais do mercado, oferecendo uma interface simples para cadastrar e gerenciar as informações essenciais do dia a dia do estabelecimento.
